System Programming

A collection of low-level programming examples and projects in C, exploring core concepts of **Linux system programming**.

Purpose

This repository is my learning journey through system programming concepts:
- Process management (`fork`, `wait`, `exec`)
- Inter-Process Communication (IPC)
- Shared memory (`mmap`, `shm_open`)
- File I/O operations
- Semaphores & synchronization
- Low-level memory management
